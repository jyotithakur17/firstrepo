GIT COMMANDS
git init : initializes a new blank git repository.
ls -lart : shows all hidden folders
git add <file_name> : to add untracked file to staging area where they are ready to commit
git add -A : to add all untracked files 
git status : shows the status of all the files whether they are untracked(not added to git), 		      staged(is ready to commit) or commit is done(nothing to commit)
touch <file_name> : to create blank files
git commit : it will first open another pane where we have to write the commiting message and 	        then it will commit
